## Attach Elastic an Network Interface to EC2 Instance

On this demo, we will Attach an elastic network interface to an existing EC2 instance. On the AWS dashboard, go to the search bar and type ``EC2`` then click on the first service that appears.

<img width="943" height="178" alt="aws-attach-an-elastic-network-interface-on-ec2" src="https://github.com/user-attachments/assets/78c95758-672a-484c-8033-f979e9588ccb" />
<br>
<br>

on the left-panel, click ```Instances```.

<img width="233" height="437" alt="2026-04-07_14-54" src="https://github.com/user-attachments/assets/7be760ad-f9e3-41ea-9d82-d63725d098c8" />
<br>
<br>

Click the running instance. Then navigate to ```Actions > Networking > Attach network interface```.

<img width="580" height="348" alt="2026-04-07_14-55" src="https://github.com/user-attachments/assets/76ae20a3-675d-408c-8ec9-e246a2e14dba" />
<br>
<br>

We will then select the vpc and interface then click on ```Attach```.
