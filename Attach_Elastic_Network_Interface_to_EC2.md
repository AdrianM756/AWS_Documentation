## Attach Elastic Network Interface to EC2

On this demo, we will attach an elastic network interface on an existing EC2 instance. On the search bar, type ``eC2`` then select ``EC2``.

<img width="940" height="182" alt="elastic-1" src="https://github.com/user-attachments/assets/e90426f2-75e2-4d48-8811-db23ca5d5d11" />
<br>
<br>

Navigate to ``Instance`` and click on the EC2 instance.

<img width="944" height="261" alt="elastic-interface-2" src="https://github.com/user-attachments/assets/4f0f1957-f410-4826-a093-08d31a93bf96" />
<br>
<br>

Click on <b>Actions</b> > <b>Networking</b> > <b>Attach network interface</b>.

<img width="442" height="241" alt="elastic-interface-3" src="https://github.com/user-attachments/assets/9d145804-46e6-4cc3-843a-249de61591d5" />
<br>
<br>

Select the network interface and then click ``attach``.
