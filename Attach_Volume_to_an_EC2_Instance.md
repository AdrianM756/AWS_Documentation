## Attach Volume to EC2 Instance

On this demo, we will Attach a volume to an EC2 Instance. On the search bar, type ```volumes``` then select the one that is under ```EC2 feature```.

<img width="821" height="337" alt="2026-04-19_22-54" src="https://github.com/user-attachments/assets/41cf7dea-3e4c-4da5-8ade-6106bbc8535d" />
<br>

We will then select the existing volume available(on the instance, it is the ```devops-volume```.)

<img width="818" height="178" alt="2026-04-19_22-56" src="https://github.com/user-attachments/assets/4981095f-3322-4125-ad13-21678c769299" />
<br>

Under ```Actions```, select ```Attach volume```.

<img width="295" height="125" alt="2026-04-19_22-57" src="https://github.com/user-attachments/assets/39ea5abe-c1bf-4cb0-9c33-144ba4fa28e7" />
<br>

We will then select the EC2 instance where we will attach the volume(on this case, it's the ```devops-ec2```). For the device name, we will input ```/dev/sdb``` then click ```Attach volume```.

<img width="1550" height="544" alt="2026-04-19_22-58" src="https://github.com/user-attachments/assets/cca450fb-7e7e-4c5b-b929-47eacd82cf43" />
