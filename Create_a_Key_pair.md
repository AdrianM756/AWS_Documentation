## Create a Key pair

On this demo, we will create an aws key pair.
<br>

on the search bar, type and click on **key pairs**.

<img width="937" height="187" alt="image" src="https://github.com/user-attachments/assets/b1f89f5c-da7b-4081-b0be-0279a7614620" />

<img width="1590" height="187" alt="2026-01-10_11-05" src="https://github.com/user-attachments/assets/f71919a4-4c91-4f7f-8523-bf579dd60439" />
