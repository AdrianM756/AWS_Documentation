## GP3 Volume

Amazon [GP3](https://aws.amazon.com/ebs/volume-types/) volumes are the latest generation of [general-purpose SSD-based EBS volumes](https://aws.amazon.com/ebs/general-purpose/) that enable customers to provision performance independent of storage capacity, while providing up to 20% lower pricing per GB than existing GP2 volumes. 
<br>

## Create GP3 Volume

In this demo, we will create a GP3 Volume. On the Search bar, type **Volumes** and select **Volumes** with the **EC2 feature**.
