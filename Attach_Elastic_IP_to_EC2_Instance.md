## Attach Elastic IP to EC2 Instance

On this demo, we attach an Elastic IP to an EC2 Instance in AWS.
<br>

On the search bar, type ```EC2``` and click on ```EC2```.

<img width="940" height="182" alt="elastic-1" src="https://github.com/user-attachments/assets/6007997c-46b4-4316-9688-d525100326e9" />
<br>

Under ```Resources```, select ```Elastic IPs```.

<img width="325" height="189" alt="elastic-2" src="https://github.com/user-attachments/assets/c2f7eeef-05e8-415c-8291-bf77fad6b980" />
<br>

Click on the Elastic IP.

<img width="995" height="189" alt="elastic-3" src="https://github.com/user-attachments/assets/f893a819-298a-4d02-ae3c-798da4295581" />
<br>

Select the ```Associate Elastic IP Address```.

<img width="1549" height="104" alt="elastic-4" src="https://github.com/user-attachments/assets/ed8d9dca-2341-4866-ae4b-81d363d8ee27" />
<br>

Select the Instance then click ```Associate```.

<img width="1682" height="572" alt="elastic-5" src="https://github.com/user-attachments/assets/77d7a7ad-1901-4f6b-a4fb-358e393d9481" />
