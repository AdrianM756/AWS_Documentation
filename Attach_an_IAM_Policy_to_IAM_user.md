## Attach an IAM Policy to IAM user

On the AWS Console Search bar, type ```IAM``` and select the first service that appears.

<img width="743" height="283" alt="image" src="https://github.com/user-attachments/assets/c8c04572-4f2a-4994-a8aa-86e20427e101" />
<br>
<br>

Under ```Access Management```, select ```IAM user``` and select the user ```iamuser_rose```.

<img width="218" height="209" alt="image" src="https://github.com/user-attachments/assets/c184d93e-a8e5-43f3-ac62-a50ffbdea893" />
<br>

<img width="213" height="64" alt="image" src="https://github.com/user-attachments/assets/bc8ee09e-91fc-49ca-b73c-37ec7bb8d598" />
<br>
<br>

Under Permissions, select ```Add permissions```.

<img width="1313" height="262" alt="image" src="https://github.com/user-attachments/assets/29a40152-59b8-4c40-b377-166c19473608" />
<br>
<br>

Select the ```Attach policies directly```. Search and select the policy ```iampolicy_rose``` then click ```Next```.

<img width="1608" height="416" alt="image" src="https://github.com/user-attachments/assets/9be5c4b4-46ca-4d24-ba1c-04f7c1136a71" />
<br>
<br>

On the Review section, click ```Add permission```.
